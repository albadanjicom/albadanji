# MR Newsletter 자동화 파이프라인

다른 PC로 폴더를 옮겨서 실행하기 위한 가이드입니다. 환경 설정부터 실행까지 아래의 순서대로 진행해 주세요.

## 1. 사전 준비 사항 (Prerequisites)

새로운 PC에 다음 프로그램들이 설치되어 있어야 합니다.

1. **Python 설치**
   - [Python 공식 홈페이지](https://www.python.org/downloads/)에서 Python 3.9 이상의 버전을 다운로드하여 설치합니다.
   - **중요:** 설치 화면 하단의 **`Add Python to PATH`** (또는 `Add python.exe to PATH`) 체크박스를 반드시 체크하고 설치해야 합니다!
2. **Chrome 브라우저 설치**
   - Selenium을 이용한 크롤링/자동화를 위해 [Google Chrome](https://www.google.co.kr/chrome/)이 설치되어 있어야 합니다.

## 2. 라이브러리 설치

이 폴더로 이동한 후, 필요한 파이썬 패키지(모듈)들을 설치해야 합니다.

1. 폴더 주소창에 `cmd`를 입력하고 엔터를 눌러 커맨드 프롬프트를 엽니다 (또는 터미널을 열어 이 폴더로 이동).
2. 아래 명령어를 입력하여 필요한 패키지를 모두 설치합니다:
   ```bash
   pip install -r requirements.txt
   ```

## 3. 환경 변수 설정 (.env 파일 생성)

자동화에 필요한 계정 정보나 비밀번호 등은 암호화되거나 별도로 관리되어야 합니다.

1. 이 폴더 안에 있는 `.env.example` 파일을 복사합니다.
2. 복사된 파일의 이름을 **`.env`** 로 변경합니다.
3. `.env` 파일을 메모장이나 코드 편집기로 열고, 파일 안에 있는 항목들을 새로운 PC 상황이나 실제 사용할 계정에 맞게 값을 채워 넣습니다:
   - `SMTP_...`: 이메일 발송에 사용할 구글 이메일 및 앱 비밀번호
   - `NAVER_ID`, `NAVER_PW`: 크롤링을 위한 네이버 계정 정보
   - `GITHUB_...`: 웹사이트 배포를 위한 깃허브 정보

## 4. 실행 방법

모든 세팅이 끝났습니다! 이제 아래의 방법으로 스크립트를 실행할 수 있습니다.

### 방법 1: 배치 파일로 간단히 실행 (추천)
- 폴더 내에 있는 **`run_daily.bat`** 파일을 더블클릭하여 실행합니다.
  *(참고: 어느 위치/폴더로 옮기더라도 정상 동작하도록 내부 경로가 업데이트 되어 있습니다.)*

### 방법 2: 명령 프롬프트(CMD)에서 직접 실행
- 이 폴더 위치에서 명령 프롬프트를 열고 아래 명령어를 입력합니다:
  ```bash
  python execution\run_daily.py
  ```

## 5. 자주 발생하는 문제 (Troubleshooting)

- **`'pip'은(는) 내부 또는 외부 명령... 아닙니다.` 에러가 날 경우**: Python 설치 시 `Add Python to PATH`를 체크하지 않아서 발생합니다. Python을 삭제하고 체크박스를 찾아 다시 설치해 주세요.
- **크롬 드라이버 에러**: 설치된 Chrome 브라우저의 버전과 호환성 문제일 수 있습니다. `webdriver-manager`가 자동으로 버전을 맞추도록 되어 있지만, 오류가 지속되면 Chrome 브라우저를 최신 버전으로 업데이트하세요.
- **네이버 크롤링 로그인 실패 / 이메일 발송 실패**: `.env` 파일에 아이디, 비밀번호(앱 비밀번호)가 오타 없이 정확하게 입력되었는지 다시 확인해 주세요.
