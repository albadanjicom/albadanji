# 💎 알바단지(AlbaDanji) 뉴스레터 시스템 가이드

이 프로젝트는 파이썬 기반의 웹 크롤링 및 뉴스레터 자동화 파이프라인입니다. 
다른 PC로 폴더를 옮겨서 실행하거나 새로운 환경에서 시작할 때, 아래의 **[보안 지침]**과 **[설정 가이드]**를 반드시 따라주세요.

---

## 🔒 0. 치명적인 보안 수칙 (Critical Security)

> [!CAUTION]
> **계정 보안 및 유출 방지를 위해 다음 내용을 반드시 숙지하세요.**

1. **`.env` 파일은 절대 공유하지 마세요**: 비밀번호, API 토큰 등이 담긴 `.env` 파일은 깃허브 등 공개된 장소에 절대 올리지 마십시오. (이미 `.gitignore`에 등록되어 자동으로 제외되도록 설정되어 있습니다.)
2. **앱 비밀번호(App Password) 사용**: 구글 계정의 실제 비밀번호 대신, [구글 보안 설정]에서 생성한 16자리 '앱 비밀번호'만 사용하세요.
3. **깃 기록(Git History) 주의**: 코드 내에 비밀번호를 직접 입력(하드코딩)하고 커밋하면, 나중에 지우더라도 기록에 남아 해킹의 타겟이 됩니다. 반드시 `.env` 환경 변수를 사용하세요.
4. **보안 토큰 인증**: 구글 앱스 스크립트와 통신할 때 사용되는 `GAS_API_TOKEN`은 외부인의 접근을 차단하는 열쇠입니다. 이 토큰이 유출되면 즉시 변경해야 합니다.

## 🏗️ 1. 새로운 PC 환경 설정 (Setup Guide)

새로운 PC에서 시스템을 구동하려면 다음 과정을 순서대로 진행하세요.

### 1-1. 필수 프로그램 설치
1. **Python 설치 (3.9 이상)**
   - [python.org](https://www.python.org/downloads/)에서 설치 시 **`Add Python to PATH`** 체크박스를 반드시 선택하세요.
2. **Chrome 브라우저 설치**
   - 최신 버전의 구글 크롬이 설치되어 있어야 크롤링이 가능합니다.

### 1-2. 라이브러리 및 의존성 설치
폴더 내에서 터미널(또는 CMD)을 열고 아래 명령어를 입력하여 필요한 패키지를 설치합니다.
```bash
pip install -r requirements.txt
```

### 1-3. 환경 변수(.env) 복사 및 설정
1. `.env.example` 파일을 복사하여 **`.env`** 파일을 만듭니다.
2. 이전 PC에서 사용하던 `.env` 내용이 있다면 그대로 옮겨 적거나, 새로운 계정 정보를 입력합니다.
   - `GMAIL_APP_PASSWORD`: 새로운 계정의 앱 비밀번호
   - `GAS_API_TOKEN`: 시스템 보안을 위한 고유 토큰 (임의의 긴 문자열)
   - `ADMIN_EMAIL`: 발행 결과를 보고받을 관리자 이메일

## 🤖 2. 실행 및 운영 방법

### 2-1. 네이버 로그인 세션 만들기 (최초 1회)
네이버 카페 크롤링을 위해 최초 1회 로그인이 필요합니다.
```bash
python execution/setup_login.py
```
창이 뜨면 네이버에 로그인하세요. 로그인이 완료되면 자동으로 창이 닫히며 세션(쿠키)이 저장됩니다.

### 2-2. 데일리 자동화 실행
```bash
python execution/run_daily.py
```
- 고정 공고 동기화 -> 웹 크롤링 -> 뉴스레터 빌드 -> 이메일 발송 -> 웹사이트 배포(Git Push) 순으로 진행됩니다.

---

## 📂 3. 프로젝트 구조 (Structure)
- `execution/`: 크롤러, 발송기, 동기화 스크립트 등 핵심 로직
- `newsletter-website/`: 사용자에게 보여지는 웹사이트 소스
- `templates/`: 뉴스레터 이메일 HTML 템플릿
- `data/`: 수집된 공고가 저장되는 SQLite 데이터베이스

---

## 🆘 4. 문제 해결 (Troubleshooting)

- **'Unauthorized' 에러 발생**: `.env`의 `GAS_API_TOKEN`과 구글 앱스 스크립트에 설정된 토큰 값이 일치하지 않을 때 발생합니다.
- **크롤링 실패**: 네이버 세션이 만료되었을 수 있습니다. `setup_login.py`를 다시 실행해 주세요.
- **이메일 발송 실패**: 구글 계정의 앱 비밀번호가 유효한지, 혹은 `.env`에 오타가 없는지 확인하세요.
